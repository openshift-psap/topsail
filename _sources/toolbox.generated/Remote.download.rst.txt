remote download
===============

Downloads a file in a remote host




Parameters
----------


``source``  

* The URL of the file to download


``dest``  

* The location where to save the file


``force``  

* Force the download


``executable``  

* If true, make the file executable


``tarball``  

* If true, untar the tarball


``zip``  

* If true, unzip the zipball (currently only on MacOS)
