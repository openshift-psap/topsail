llmd deploy_gateway
===================

Deploys a GatewayClass and Gateway object

Default gateway class is created by the DSCI -> GatewayConfig/default-gateway -> GatewayClass/data-science-gateway-class


Parameters
----------


``name``  

* Name of the gateway to deploy

* default value: ``openshift-ai-inference``


``gateway_class``  

* Name of the gateway class to deploy

* default value: ``data-science-gateway-class``


``namespace``  

* Namespace where the gateway will be deployed

* default value: ``openshift-ingress``
